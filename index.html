Write a social media carousel about: [YOUR TOPIC]

Use EXACTLY this format:

[Main title for the carousel]

1. [First point title]
[2-3 sentences explaining this point]

2. [Second point title]
[2-3 sentences explaining this point]

3. [Third point title]
[2-3 sentences explaining this point]

...continue for 5-8 points total.

Rules:
- Start with ONE title line (no numbering)
- Each point: number + period + title on one line
- Body text on the next lines (2-3 sentences)
- Separate each point with a blank line
- No emojis, no bold, no markdown
- Keep each point under 40 words
- End with a final point, no CTA needed
